FEWTD 是一个轻量级、**即插即用 (Plug-and-Play)** 的通用解码器模块。本项目训练方案参考了 [EMCAD (ECCV 2024)](https://github.com/SLDGroup/EMCAD)，旨在提升计算机视觉任务中特征融合与上采样的性能。
